Project : Food Donation Website : SOLUTION CHALLENGE

  • Source code

This project involves developing Prajaahar Rakshak, a Node.js, Express, EJS, Bootstrap, and MongoDB system to streamline surplus food donations from donors to NGOs/food banks. The system implements dynamic agent assignment, real-time updates, and a responsive design, optimizing the donation process and ensuring efficient coordination.

Prajaahar Rakshak aims to reduce food wastage by creating a seamless connection between food donors and organizations that distribute food to those in need. The platform enhances the efficiency and effectiveness of food donation logistics.

  • Utilizes Node.js, Express, EJS, Bootstrap, and MongoDB
  • Implements dynamic agent assignment and real-time updates
  • Optimizes food donation process with a responsive design
