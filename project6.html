Project : Full-Stack Contact List

  • Source code

This project involves developing a full-stack contact list application using React for the frontend and Node.js, Express, and MongoDB for the backend. The application allows users to create, read, update, and delete contact information. It features user authentication and a responsive design.

The Full-Stack Contact List aims to provide a simple and efficient way for users to manage their contacts. It leverages modern web development technologies to deliver a seamless user experience.

  • Utilizes React, Node.js, Express, and MongoDB
  • Allows users to create, read, update, and delete contacts
  • Features user authentication and a responsive design
