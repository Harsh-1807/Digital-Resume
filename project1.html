Project : GIS Based Risk Factor Prediction

  • Source code

This project involves developing GeoRiskAccess, a web-based tool using Python, Excel, Flask, and Gemini API. It integrates GIS and Google Earth Engine data to assess urban risk factors in Indian cities. The tool utilizes a Random Forest classifier to analyze real-time data on population density, air quality, traffic, land use, and more. It offers precise risk evaluations and actionable visualizations to promote urban sustainability and resilience.

GeoRiskAccess aims to provide city planners and decision-makers with a comprehensive tool to assess and mitigate urban risks effectively. By combining various data sources and advanced machine learning techniques, the project delivers valuable insights into urban safety and development.

  • Integrates GIS and Google Earth Engine data
  • Utilizes a Random Forest classifier for real-time data analysis
  • Provides actionable visualizations for urban risk evaluations
