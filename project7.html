Project: AI Tour Itinerary Generator

  • Source Code

The AI Tour Itinerary Generator is a web-based application that leverages AI to generate personalized travel itineraries. Built using Python, Streamlit, and the OpenAI API, this project provides users with customized travel plans based on their preferences and inputs.

Key Features:

  • Uses AI to generate personalized travel itineraries.
  • Built with Python and Streamlit for a seamless user experience.
  • Integrates the OpenAI API to enhance itinerary customization.
  • Provides real-time updates and recommendations.

This project was developed as part of my work with GDSC VIT PUNE, where it won the TechFesta Hackathon.
